Analyzing ADHD and its effects on academic performance

This data analysis project aims to determine if ADHD/ADD status and internet usage in learning affect a student's parent-reported academic performance, and to predict academic performance based off of these given factors via a multiclass classification model.

Working with two other group members, I handled the bulk of the data cleaning and exploratory analysis as well as assembling the final report document.

This project was done entirely in Python via Jupyter Notebooks, with large datasets provided by the National Household Education Surveys Program.
